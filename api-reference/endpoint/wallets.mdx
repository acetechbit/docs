# Overview
This section provides details on how to manage crypto and fiat wallets using the Wallets API.

## Authentication
All endpoints require authentication using a Bearer Token.

# Endpoints

## Create Multiple Wallets
**Endpoint:**  
`POST /wallets/create`

**Description:**  
Creates multiple wallets for a user.

**Security:**  
Requires Bearer Token authentication.

**Responses:**

```json
{
  "success": true,
  "message": "Wallets created successfully",
  "wallets": [
    {
      "walletId": "btc-wallet-abc123",
      "asset": "BTC",
      "balance": "0.0325"
    }
  ]
}
```

---

## Create Wallet for a Specific Asset
**Endpoint:**  
`POST /wallets/create/{asset}`

**Description:**  
Creates a wallet for a specific asset (e.g., BTC, ETH, TRX).

**Security:**  
Requires Bearer Token authentication.

**Parameters:**

- `asset` (string) - The cryptocurrency asset (e.g., BTC, ETH, TRX).

**Responses:**

```json
{
  "success": true,
  "wallet": {
    "walletId": "eth-wallet-xyz789",
    "asset": "ETH",
    "balance": "1.5000"
  }
}
```

---

## Get All Wallets
**Endpoint:**  
`GET /wallets`

**Description:**  
Retrieves all wallets (crypto and fiat) for the authenticated user.

**Security:**  
Requires Bearer Token authentication.

**Query Parameters:**

_None_

**Responses:**

```json
{
  "cryptoWallets": [
    {
      "id": 1,
      "walletId": "btc-wallet-abc123",
      "asset": "BTC",
      "balance": "0.0325",
      "address": "67423eb2159cfe1ab70b31ea32fb781b",
      "createdAt": "2023-11-23T20:44:36.008Z",
      "updatedAt": "2023-11-23T23:09:14.810Z"
    }
  ],
  "fiatWallets": [
    {
      "id": 1,
      "currency": "USD",
      "balance": 500.50,
      "userId": 101,
      "createdAt": "2023-10-01T12:00:00.000Z",
      "updatedAt": "2023-10-05T08:15:00.000Z"
    }
  ]
}
```
